## 🚀 Overview
A **Machine Learning-powered web application** built with **Python, Flask, and Random Forest Regressor** to predict house prices based on key features like:
- Square Footage
- Bedrooms
- Bathrooms
- Floors
- Waterfront Presence
- House Condition
